Burnlist

Create a mood-based playlist from your liked songs and playlists you own (your latest playlists when you have three or more). Next removes a track immediately; ~20 seconds only if you keep playing the same track — then the whole list disappears from your library when it’s empty.

How it works

  1. Pick a feeling or theme and size — we pull from likes plus your playlists (top 3 owned if you have 3+, else all owned).
  2. Log in with Spotify; keep this tab open while you listen.
  3. Open the playlist on any device. Next removes the current song right away; ~20s on a song only if you don’t skip. When it’s empty, the playlist leaves your library.

Tips

  • More likes = more variety for each mood.
  • Same mood twice usually picks different songs (random order).
  • If something won’t play, skip — the row still burns off.
